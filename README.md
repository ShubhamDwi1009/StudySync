# StudySync

StudySync is a full-stack student productivity web app built for a hackathon-style demo. It combines a polished dashboard, a planner, session tracking, persistent focus timing, analytics, and lightweight gamification in one local app.

## Stack

- Node.js 24
- Built-in `node:sqlite` for local persistence
- Vanilla HTML, CSS, and JavaScript SPA frontend

## Features

- Dashboard with greeting, quote, study stats, progress circle, streak, and quick actions
- Subject management with weak-subject highlighting
- Daily task planner and simple seven-day timetable board
- Study session history with manual logging
- Persistent focus timer with start, pause, resume, autosave, and cross-view continuity
- Progress analytics for weekly hours, tasks, streak heatmap, subject momentum, and badges
- Settings for student name, weekly goal, and default focus duration

## Run

```bash
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test
```
